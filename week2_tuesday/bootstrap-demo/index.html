Användbarhet

Användbarhet är hur lätt det är att använda och lära sig använda ett objekt som är skapat av människor. Det innebär att en webbplats eller applikation ska vara enkel att använda, effektiv och tillfredsställande för användarna.

Responsiv Webbdesign

Responsiv webbdesign innebär att webbplatser fungerar bra oavsett om de besöks via en dator, mobil eller surfplatta. CSS och media queries används för att anpassa layouten beroende på skärmens storlek.

Bootstrap

Bootstrap är ett frontend ramverk som används för att enkelt skapa responsiv design. Det innehåller ett antal CSS-stylesheets och JavaScript-filer som underlättar webbdesign och gör det möjligt att snabbt skapa moderna och funktionella webbsidor.
